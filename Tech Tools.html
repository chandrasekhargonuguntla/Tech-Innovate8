Tech Tools

Developer Tools:

Developer Tools

(or "development tools" or short "DevTools") are programs that allow a developer to create, test and debug software.

Current browsers provide integrated developer tools, which allow to inspect a website. They let users inspect and debug the page's HTML, CSS, and JavaScript, allow to inspect the network traffic it causes, make it possible to measure its performance, and much more.

Learning and Education

    Online Courses and Tutorials

  • Udemy: Courses on various programming languages and technologies
  • Coursera: University-style courses on computer science and software development.
  • edX: Offers online courses from institutions like MIT and Harvard.

    Documentation and Reference

  • W3Schools: Tutorials and references on web development languages
  • JavaScript.info: In-depth guide and tutorials on JavaScript.
  • MDN Web Docs: Comprehensive resource for web development, including HTML, CSS, and JavaScript

Code Editors and IDEs

  • Visual Studio Code: Highly customizable, supports many languages and extensions.
  • Sublime Text: Lightweight and fast code editor.
  • Atom: Open-source code editor with a focus on modularity.
  • IntelliJ IDEA: Powerful IDE, particularly strong for Java, Kotlin, and other JVM languages.

Version Control Systems

  • Git: Distributed version control system, essential for modern development.
  • GitHub: Hosting service for Git repositories with collaboration features.
  • GitLab: Integrated CI/CD platform with Git repository management.
  • Bitbucket: Git repository hosting with built-in CI/CD.

Testing Tools

Database Management Tools

  • Jest: JavaScript testing framework with a focus on simplicity.
  • Mocha: Flexible testing framework for JavaScript.
  • JUnit: Popular testing framework for Java.
  • Selenium: Tool for automating web browsers for testing purposes.
  • pgAdmin: Management tool for PostgreSQL databases.
  • MongoDB Compass: GUI for managing MongoDB databases.
  • MySQL Workbench: Visual tool for MySQL database design, development, and administration.

Productivity Tools

Productivity tools are applications designed to help individuals and organizations improve their work efficiency and effectiveness. They streamline workflows, enhance collaboration, and optimize time management to empower individuals and teams to achieve their goals with less effort and in less time.

The right tech tool can make all the difference when it comes to productivity, and nobody knows that better than tech industry leaders. We’ve all heard the phrase, “There’s an app for that,” but it’s still surprising how many daily tasks and needs can be addressed with technology (often for free or at minimal cost).

Design Tools

Design tools are essential for creating everything from graphics and websites to presentations and prototypes. Whether you're a graphic designer, web developer, or just looking to create visually appealing content, here's a comprehensive list of design tools that cater to different needs and skill levels

Graphic Design

  • Adobe Photoshop Industry-standard tool for photo editing, digital painting, and graphic design.
  • Adobe Illustrator Vector graphic editor for creating logos, illustrations, and complex designs.
  • CorelDRAW A vector graphic design tool known for its robust features and user-friendly interface.
  • Affinity Designer A professional graphic design tool that's a popular alternative to Adobe Illustrator.
  • GIMP (GNU Image Manipulation Program) A free, open-source alternative to Photoshop with a wide range of features.

UI/UX Design

  • Figma A web-based design tool for UI/UX design with real-time collaboration features.
  • Adobe XD A tool for designing and prototyping user experiences for web and mobile apps.
  • Sketch A vector-based design tool popular for UI/UX design, especially in macOS environments.
  • InVision Provides prototyping, collaboration, and workflow tools for design projects.

Prototyping

  • Balsamiq A tool focused on low-fidelity wireframing to help brainstorm and layout concepts quickly.
  • Axure RP Advanced prototyping tool with interactive elements and dynamic content capabilities.
  • Marvel App Allows for wireframing, prototyping, and user testing with an easy-to-use interface.

Security Tools

Security tools are essential for protecting your digital assets, data, and systems from threats such as malware, unauthorized access, and cyberattacks. Here's an overview of key security tools and their features to help safeguard your environment.

Kali Linux is a Linux distribution designed for digital forensics and penetration testing. It is maintained and funded by Offensive Security. The software is based on the Debian Testing branch: most packages Kali uses are imported from the Debian repositories. wikipedia

The Metasploit Project is a computer security project that provides information about security vulnerabilities and aids in penetration testing and IDS signature development. It is owned by Boston, Massachusetts-based security company, Rapid7. wikipedia

John the Ripper is a free password cracking software tool. Originally developed for the Unix operating system, it can run on fifteen different platforms. wikipedia

Wireshark is a free and open-source packet analyzer. It is used for network troubleshooting, analysis, software and communications protocol development, and education. Originally named Ethereal, the project was renamed Wireshark in May 2006 due to trademark issues. wikipedia

Nikto is a free software command-line vulnerability scanner that scans web servers for dangerous files/CGIs, outdated server software and other problems. It performs generic and server type specific checks. It also captures and prints any cookies received. wikipedia
